# Technician App

Flutter Mobile App für Techniker zur Bearbeitung von Checklisten, auch offline.

## Funktionen
- Login
- Checkliste auswählen
- Offline ausfüllen
- Fotos, Kommentare, Unterschrift hinzufügen
- Synchronisieren bei Verbindung
- Checkliste einreichen

## Tech Stack
- Flutter
- Dart
- SQLite / Drift (Offline-Speicher)

## Setup

```bash
flutter pub get
flutter run
```
