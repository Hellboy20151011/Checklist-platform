# Workflow

## Statusmodell

- DRAFT
- IN_PROGRESS
- SUBMITTED
- IN_REVIEW
- APPROVED
- REJECTED
- REWORK_REQUIRED

## Standardablauf

1. Admin erstellt Vorlage
2. Techniker startet Checklisten-Instanz
3. Techniker bearbeitet Checkliste
4. Techniker reicht Checkliste ein
5. Prüfer kontrolliert
6. Prüfer gibt frei oder sendet zurück

## Spätere Regeln
- Kritischer Mangel => Eskalation
- Fehlendes Pflichtfoto => Rückgabe
- Keine Abweichung => automatische Freigabe möglich
