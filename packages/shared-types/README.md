# shared-types

Gemeinsame TypeScript-Typdefinitionen für die Checklist Platform.

## Enthält
- Enums: `UserRole`, `TemplateStatus`, `ChecklistStatus`, `WorkflowAction`, `TemplateItemType`
- Interfaces: `User`, `ChecklistTemplate`, `TemplateItem`, `ChecklistInstance`, `InstanceAnswer`, `WorkflowEvent`, `SyncChangeLog`

## Verwendung

```typescript
import { UserRole, ChecklistStatus, ChecklistTemplate } from '@checklist-platform/shared-types';
```
